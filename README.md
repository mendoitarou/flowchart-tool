# Flowchart-Draw-Tool
JIS X0121に準拠したフローチャートをコードベースで書けるツール

Draw.io や Mermaid では再現しにくい、**日本産業規格（JIS X 0121）準拠のフローチャート**をテキスト（DSL）で記述・描画するWebツールです。

> [!WARNING]
> このツールは現在制作中です。
> 現状、多くの不具合が存在します。
> 規格に準拠しない、不正確なフローチャートが出力される可能性があります。ご注意ください。
> (すでにサンプルとして提示しているフローチャートが思いっきりバグっています。)

> [!WARNING]
> 利便性向上のためにJIS規格に無い記号が実装されています。
> そのため、ほぼ準拠となってしまいました。ご了承ください。

## 日本産業規格
規格番号: JIS X0121

規格名称: 情報処理用流れ図・プログラム網図・システム資源図記号

## 特徴

- **日本語キーワード DSL** — `端子`、`処理`、`判断`、`入出力` など馴染みのある用語でフローを記述
- **全13種類の図形** — 端子・処理・判断・入出力・結合子・ページ外結合子・定義済み処理・手操作入力・手動操作・並行処理・書類・注釈・初期設定/初期条件
- **自動レイアウト** — 上から下への階層レイアウトを自動計算
- **ループ対応** — 後戻りエッジを自動で左側に迂回描画
- **エクスポート** — SVG / PNG 形式で書き出し可能
- **リアルタイムプレビュー** — 入力に合わせて自動更新（デバウンス付き）

## 使い方

1. [https://flowchart-draw-tool.mudana.work](https://flowchart-draw-tool.mudana.work) をブラウザで開く
2. 左ペインに DSL コードを入力（または「サンプル」ボタンを押す）
3. 自動的にプレビューが更新される（手動描画は「描画」ボタンまたは `Ctrl+Enter`）
4. 「SVG保存」「PNG保存」ボタンで書き出し
